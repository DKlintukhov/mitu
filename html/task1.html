Моя первая веб-страница

HTML (HyperText Markup Language) — это язык разметки, который используется для создания структуры веб‑страниц.
Он позволяет определять элементы контента: заголовки, абзацы, списки, изображения, ссылки и другие блоки, формируя основу визуального представления информации в браузере.

В отличие от языков программирования (таких как JavaScript, Python или Java), HTML не выполняет вычислений и не содержит логики обработки данных. Его задача — исключительно разметка и структурирование контента. Для добавления интерактивности и динамического поведения к HTML подключают CSS (для стилизации) и JavaScript (для программирования поведения).
