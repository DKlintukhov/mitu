# Запуск проекта

## 1. Установите NodeJS
Скачайте и установите NodeJS с официального сайта: https://nodejs.org/en/download.

## 2. Перейдите в директорию сервера
```
cd mitu/js/async
```

## 3. Запустите сервер
```
node script.js
```

## 4. Запустите клиент
Лучший способ — через Live Server в VS Code:

1. Установите расширение Live Server в VS Code.
2. Откройте файл mitu/index.html.
3. Нажмите кнопку Go Live внизу справа или кликните правой кнопкой по файлу → Open with Live Server.
4. В браузере откроется страница http://localhost:5500.
